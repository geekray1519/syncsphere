# Google Play ストア掲載情報

## アプリ名
SyncSphere - ファイル同期

## 短い説明（80文字以内）
無料のファイル同期アプリ。ローカル・LAN・リモートサーバー間でかんたん同期。

## 詳しい説明

📁 **SyncSphere** は、ファイルとフォルダの同期を誰でも簡単に行えるアプリです。

🔄 **3つの同期モード**
• ミラー同期 — 元フォルダの内容を先フォルダに完全コピー
• 双方向同期 — 両方のフォルダの変更を互いに反映
• アップデート同期 — 新しいファイルのみをコピー（削除なし）

🌐 **豊富な接続方法**
• ローカルフォルダ / USBドライブ
• LAN（同じWi-Fiネットワーク内）
• SFTP（リモートサーバー）
• FTP / FTPS
• P2P（デバイス間直接接続）

✨ **使いやすさを重視**
• わかりやすいセットアップウィザード
• QRコードでかんたんデバイス接続
• 日本語完全対応
• ダークモード対応
• リアルタイムの同期進捗表示

🛡️ **安心のデータ保護**
• TLS暗号化通信
• バージョン管理（ゴミ箱方式 / タイムスタンプ方式）
• 競合ファイルの自動検出と手動解決
• ファイルデータは一切外部に送信されません

⚡ **高性能**
• ファイルの差分のみを転送
• 並列ファイルコピー
• ファイル変更の自動検出（リアルタイム同期）
• フィルタ設定で不要ファイルを除外

📊 **同期管理**
• 複数の同期ジョブを作成・管理
• スケジュール設定（定期実行 / リアルタイム）
• 詳細な同期ログと統計
• バッチ実行対応

💡 **こんな方におすすめ**
• PCとスマホのファイルを同期したい
• USBドライブに定期バックアップしたい
• 複数のPCでドキュメントを共有したい
• リモートサーバーとファイルを同期したい
• 写真やドキュメントの自動バックアップがほしい

🆓 **完全無料**
すべての機能を無料でご利用いただけます。広告が表示されますが、動画広告を見ることで一時的に非表示にできます。

📱 **対応環境**
• Android 7.0以上
• Windows 10以上（デスクトップ版もご利用いただけます）
