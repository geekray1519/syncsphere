Privacy Policy

SyncSphere ("the App") respects user privacy and is committed to protecting personal information.

1. Information We Collect

1.1 File Sync Data

The App accesses file metadata (names, sizes, modification dates) in user-specified folders for sync functionality.

File contents are never sent to our servers. All synchronization occurs directly between devices.

1.2 Device Information

  • Device identifier (randomly generated ID)
  • Device name
  • Operating system type and version
  • Network connection info (for LAN device discovery)

1.3 Advertising Data

The App uses Google AdMob to display advertisements. AdMob may collect:

  • Advertising ID
  • Approximate location
  • App usage data
  • Device information

For details, see Google's Privacy Policy.

2. How We Use Information

  • Provide and improve file sync services
  • Establish device connections
  • Display advertisements (to maintain free service)
  • Fix bugs and improve quality

3. Data Sharing

  • User file data is never shared with third parties
  • Device identifiers are shared with Google AdMob for advertising
  • Personal information is not shared except as required by law

4. Data Storage

  • Sync job settings are stored locally on the device
  • File metadata databases are stored locally
  • No data is sent to cloud servers

5. Your Rights

  • Ad Opt-out: Disable personalized ads via Google Settings > Ads
  • Data Deletion: Uninstalling the app removes all local data
  • Permission Management: Change app permissions anytime in device settings

6. Security

  • Device-to-device communication is protected by TLS encryption
  • Device IDs are generated using cryptographically secure hash functions
  • File transfers are end-to-end encrypted

7. Children's Privacy

The App is not intended for children under 13. We do not knowingly collect information from children under 13.

8. Contact

For privacy questions: syncsphere.app@gmail.com
