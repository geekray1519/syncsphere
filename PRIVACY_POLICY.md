# プライバシーポリシー / Privacy Policy

**最終更新日: 2026年2月22日**

## SyncSphere プライバシーポリシー

### はじめに

SyncSphere（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、本アプリにおける情報の収集、使用、開示について説明します。

### 1. 収集する情報

#### 1.1 ファイル同期データ
- 本アプリはファイル同期機能を提供するため、ユーザーが指定したフォルダ内のファイル名、サイズ、更新日時などのメタデータにアクセスします。
- **ファイルの内容は本アプリのサーバーに送信されることはありません。** すべての同期はデバイス間で直接行われます。

#### 1.2 デバイス情報
- デバイス識別子（ランダム生成ID）
- デバイス名
- オペレーティングシステムの種類とバージョン
- ネットワーク接続情報（LANデバイス検出のため）

#### 1.3 広告関連データ
本アプリはGoogle AdMobを使用して広告を表示します。AdMobは以下の情報を収集する場合があります：
- 広告識別子（Advertising ID）
- おおよその位置情報
- アプリの使用状況データ
- デバイス情報

詳細については、[Googleのプライバシーポリシー](https://policies.google.com/privacy)をご参照ください。

### 2. 情報の使用目的

収集した情報は以下の目的で使用します：
- ファイル同期サービスの提供と改善
- デバイス間の接続と通信の確立
- 広告の表示（無料サービスの維持のため）
- アプリの不具合の修正と品質改善

### 3. 情報の共有

- ユーザーのファイルデータは第三者に共有されません
- 広告表示のため、Google AdMobにデバイス識別子等が提供されます
- 法律で要求される場合を除き、個人を特定できる情報を第三者と共有することはありません

### 4. データの保存

- 同期ジョブの設定はデバイスのローカルストレージに保存されます
- ファイルのメタデータ（同期データベース）はデバイスのローカルストレージに保存されます
- クラウドサーバーへのデータ送信は行いません

### 5. ユーザーの権利

- **広告のオプトアウト**: Google設定 > 広告 > 広告のパーソナライズをオフにすることで、パーソナライズド広告を無効にできます
- **データの削除**: アプリをアンインストールすることで、すべてのローカルデータが削除されます
- **権限の管理**: デバイスの設定からアプリの権限をいつでも変更できます

### 6. セキュリティ

- デバイス間の通信はTLS暗号化で保護されます
- デバイス識別子は暗号学的に安全なハッシュ関数で生成されます
- ファイル転送中のデータはエンドツーエンドで暗号化されます

### 7. 子供のプライバシー

本アプリは13歳未満の子供を対象としていません。13歳未満の子供から意図的に個人情報を収集することはありません。

### 8. 個人情報保護法（APPI）に基づく開示

日本の個人情報保護法に基づき：
- **利用目的**: 上記第2項に記載の通り
- **第三者提供**: Google AdMob（広告配信目的）
- **お問い合わせ**: syncsphere.app@gmail.com

### 9. ポリシーの変更

本プライバシーポリシーは随時更新される場合があります。重要な変更がある場合は、アプリ内で通知します。

### 10. お問い合わせ

プライバシーに関するご質問やご要望は以下までご連絡ください：

**メール**: syncsphere.app@gmail.com

# Privacy Policy (English)

**Last Updated: February 22, 2026**

## Introduction

SyncSphere ("the App") respects user privacy and is committed to protecting personal information. This Privacy Policy explains how the App collects, uses, and discloses information.

## Information We Collect

### File Sync Data
- The App accesses file metadata (names, sizes, modification dates) in user-specified folders for sync functionality.
- **File contents are never sent to our servers.** All synchronization occurs directly between devices.

### Device Information
- Device identifier (randomly generated ID)
- Device name
- Operating system type and version
- Network connection info (for LAN device discovery)

### Advertising Data
The App uses Google AdMob to display advertisements. AdMob may collect:
- Advertising ID
- Approximate location
- App usage data
- Device information

For details, see [Google's Privacy Policy](https://policies.google.com/privacy).

## How We Use Information

- Provide and improve file sync services
- Establish device connections
- Display advertisements (to maintain free service)
- Fix bugs and improve quality

## Data Sharing

- User file data is never shared with third parties
- Device identifiers are shared with Google AdMob for advertising
- Personal information is not shared except as required by law

## Data Storage

- Sync job settings are stored locally on the device
- File metadata databases are stored locally
- No data is sent to cloud servers

## Your Rights

- **Ad Opt-out**: Disable personalized ads via Google Settings > Ads
- **Data Deletion**: Uninstalling the app removes all local data
- **Permission Management**: Change app permissions anytime in device settings

## Security

- Device-to-device communication is protected by TLS encryption
- Device IDs are generated using cryptographically secure hash functions
- File transfers are end-to-end encrypted

## Children's Privacy

The App is not intended for children under 13. We do not knowingly collect information from children under 13.

## Contact

For privacy questions: syncsphere.app@gmail.com
